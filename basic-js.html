Шпаргалка:

Типы данных - во всех типов данных в JavaScript, кроме объектов, значения не могут быть модифицированы, а только перезаписаны новым полным значением. Например, в отличии от C, где строку можно посимвольно корректировать, в JavaScript строки пересоздаются только полностью. Значения таких типов называются «примитивными значениями». Примитивными значениями бывают null, undefined, boolean (true или false), а также бывают числами. Структуры данных (массивы, стек, очередь) - эти структуры данных очень хорошо помогают в программировании. К примеру стек следует принципу LIFO (Last In First Out — последним вошел, первым вышел). Если вы сложили книги друг на друга, и захотели взять самую нижнюю книгу, то сначала возьмете верхнюю, затем следующую и т.д. Очередь напоминает стек. Разница состоит в том, что очередь следует принципу FIFO (First In First Out — первым вошел, первым вышел). Когда мы стоим в очереди, то первый в ней всегда будет первым. Массивами являются спископодобными объектам. Особая структура данных, которая существует для хранения упорядоченных коллекций. Арифмитические опреации - Для выполнения различных действий над значениями (например, сложение, вычитание и т.д.) язык JavaScript предоставляет операции. Операции принимают один, два или три операнда и применяют к ним некоторые действия. В качестве операндов в зависимости от типа операции можно применять числа и строки, переменные и константы или результаты других операций. Часто используемые функции библиотеки Math - функции math довольно полезные, так как они помогают хранить различные математичексие константы и функции. В качестве примера я перечислю самые распространённые функции math, которые, по моему мнению, довольно полезные: Math.sqrt - возведение в квадрат; Math.abs - возвращает абсолютное значение числа; Math.floor - Возвращает значение числа, округлённое к меньшему целому; Math.round - возвращает значение числа, округлённое до ближайшего целого; Math.random - возвращает псевдослучайное число в диапазоне от 0 до 1. Функции lodash - библиотека функции lodash упрощает работу с массивами, числами, объектами, строками путём предоставления множества полезных методов. Благодаря этой библиотеке мы можем не тратить время на реализацию логики и алгоритмов, а просто использовать готовые методы Lodash. Логические опреации - В JavaScript есть семь логических операторов: || = (Оператор логического присваивания ИЛИ); && = (Оператор логического присваивания И); ! = (НЕ); ?? = (Оператор нулевого присваивания). Алгоритмические конструкции (условие, циклы) - при написании скриптов зачастую встаёт задача сделать однотипное действие много раз. Например, вывести товары из списка один за другим. Или просто перебрать все числа от 1 до 10 и для каждого выполнить одинаковый код. Для многократного повторения одного участка кода предусмотрены циклы. Функции (как объявить, стрелочные функции, просто function, импорт/экспорт) - для создания функций мы можем использовать объявление функции. Объявить можно через function или константу. Для того, чтобы какие-то компоненты можно было подключить и использовать в другом модуле, их надо экспортировать. Мы можем экспортировать какждый компонент по отдельности. Для этого перед определением компонента указывается ключевое слово export. А в нужном модуле нужно будет принимать экспорт, указывая ключевое слово import.
